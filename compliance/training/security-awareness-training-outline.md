## 1. Training Program Overview

### 1.1 Objectives
- Ensure all personnel understand security policies and procedures
- Reduce human-factor security risks
- Build a security-conscious culture
- Meet SOC2 and GDPR training requirements

### 1.2 Target Audience

| Group | Training Level | Frequency |
|-------|---------------|-----------|
| All Employees | General Awareness | Annually + Onboarding |
| Developers | Secure Coding | Annually |
| DevOps/Infrastructure | Security Operations | Annually |
| Management | Security Leadership | Annually |
| Privileged Users | Advanced Security | Semi-annually |

---

## 2. Core Training Modules

### Module 1: Security Fundamentals (All Employees)
**Duration:** 60 minutes  
**Format:** Online self-paced  

**Topics:**
- [ ] Why security matters (real-world breach examples)
- [ ] ALVO security policies overview
- [ ] Your role in security
- [ ] Reporting security concerns

**Learning Outcomes:**
- Understand basic security concepts
- Know company security policies
- Know how to report incidents

---

### Module 2: Password & Authentication Security
**Duration:** 30 minutes  
**Format:** Online self-paced  

**Topics:**
- [ ] Password best practices
- [ ] Multi-factor authentication (MFA)
- [ ] Password manager usage
- [ ] Recognizing credential theft attempts

**Learning Outcomes:**
- Create strong, unique passwords
- Properly use MFA
- Identify phishing for credentials

---

### Module 3: Phishing & Social Engineering
**Duration:** 45 minutes  
**Format:** Online + simulated phishing  

**Topics:**
- [ ] Types of phishing (email, SMS, voice, spear-phishing)
- [ ] Recognizing phishing indicators
- [ ] Safe email practices
- [ ] Social engineering tactics
- [ ] Reporting suspicious messages

**Practical Exercise:**
- Simulated phishing campaign (quarterly)

**Learning Outcomes:**
- Identify common phishing techniques
- Properly handle suspicious communications
- Report phishing attempts

---

### Module 4: Data Protection & GDPR
**Duration:** 45 minutes  
**Format:** Online self-paced  

**Topics:**
- [ ] What is personal data
- [ ] GDPR principles overview
- [ ] Data classification at ALVO
- [ ] Handling sensitive data
- [ ] Data subject rights
- [ ] Breach reporting obligations

**Learning Outcomes:**
- Classify data correctly
- Handle personal data appropriately
- Understand data subject rights
- Know when to report potential breaches

---

### Module 5: Secure Development (Developers)
**Duration:** 120 minutes  
**Format:** Instructor-led + hands-on  

**Topics:**
- [ ] OWASP Top 10 overview
- [ ] Secure coding practices
- [ ] Input validation
- [ ] Authentication/authorization implementation
- [ ] Secrets management
- [ ] Dependency security
- [ ] Code review for security
- [ ] Security testing (SAST/DAST)

**Learning Outcomes:**
- Write secure code
- Identify common vulnerabilities
- Use security testing tools
- Implement proper secrets management

---

### Module 6: Incident Response Awareness
**Duration:** 30 minutes  
**Format:** Online self-paced  

**Topics:**
- [ ] What constitutes a security incident
- [ ] How to report incidents
- [ ] Incident response process overview
- [ ] Your role during an incident
- [ ] Post-incident expectations

**Learning Outcomes:**
- Recognize security incidents
- Know reporting procedures
- Understand response process

---

### Module 7: Physical Security & Clean Desk
**Duration:** 20 minutes  
**Format:** Online self-paced  

**Topics:**
- [ ] Physical access controls
- [ ] Visitor management
- [ ] Clean desk policy
- [ ] Screen lock requirements
- [ ] Secure document disposal

**Learning Outcomes:**
- Follow physical security procedures
- Maintain clean desk practices
- Properly secure sensitive materials

---

### Module 8: Remote Work Security
**Duration:** 30 minutes  
**Format:** Online self-paced  

**Topics:**
- [ ] Secure home network setup
- [ ] VPN usage requirements
- [ ] Public Wi-Fi risks
- [ ] Device security
- [ ] Secure video conferencing

**Learning Outcomes:**
- Secure remote work environment
- Properly use VPN
- Protect company data while remote

## 6. Assessment & Completion

### 6.1 Assessment Requirements

| Module | Assessment Type | Passing Score |
|--------|-----------------|---------------|
| Security Fundamentals | Quiz (20 questions) | 80% |
| Password Security | Quiz (10 questions) | 80% |
| Phishing Awareness | Quiz + simulation | 80% + no clicks |
| Data Protection | Quiz (15 questions) | 80% |
| Secure Development | Quiz + code review | 80% |
| All other modules | Quiz (10 questions) | 80% |

### 6.2 Completion Requirements

- All modules must be completed within 30 days of assignment
- Failed assessments allow one retake
- Failure to complete training escalated to manager
- Repeated non-compliance escalated to HR

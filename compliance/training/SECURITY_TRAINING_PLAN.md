## 2. Training Content Outline

### Module 1: Security Fundamentals (Required)
- Why security matters
- Common threats and attack vectors
- Company security policies overview
- Your responsibilities
- How to report incidents

### Module 2: Data Protection & GDPR
- What is personal data
- GDPR principles
- Data subject rights
- Your obligations when handling data
- Data breach consequences

### Module 3: Phishing & Social Engineering
- Recognizing phishing emails
- Social engineering tactics
- What to do when suspicious
- Reporting procedures

### Module 4: Password & Access Security
- Strong password requirements
- MFA importance
- Never sharing credentials
- Secure storage of secrets

### Module 5: Secure Development (Technical Staff)
- OWASP Top 10 overview
- Input validation
- Secure authentication implementation
- Dependency management
- Code review security checklist

### Module 6: Incident Response
- How to recognize an incident
- Immediate actions to take
- Who to contact
- Evidence preservation basics

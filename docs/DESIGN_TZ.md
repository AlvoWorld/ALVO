## СТРАНИЦЫ И ЭКРАНЫ

### 1. Главный дашборд
**Что показываем:**
- Статистика системы (карточки):
  - Всего агентов
  - Работают (зеленый)
  - Желтые (ожидают)
  - Ошибки (красный)
  - Всего задач
  - Текущая модель
- График использования токенов (по времени)
- График расходов (по времени)
- Распределение моделей (круговая диаграмма)

### 2. Список агентов
**Для каждого агента:**
- Имя
- Статус (running/idle/error) — цветной индикатор
- Модель (ссылка на настройки)
- Провайдер
- Кнопки действий:
  - ▶ Запустить
  - ⏸ Пауза
  - ⚙ Настройки
  - 📊 Статистика
- Фильтры:
  - По статусу
  - По провайдеру
  - По модели
  - По приоритету

### 3. Карточка агента (модальное окно)
**Секции:**
- Основная информация
  - Имя
  - Роль
  - Статус
  - Время работы
- Настройки модели
  - Выбор модели (dropdown с поиском)
  - Выбор провайдера
  - Бесплатные модели (галочка)
- Лимиты
  - Дневной лимит токенов (ползунок/ввод)
  - Дневной лимит расходов USD (ползунок/ввод)
  - Текущее использование (прогресс-бар)
- Расписание
  - Cron выражение (ввод)
  - Активные часы (time picker)
  - Вкл/Выкл расписание
- Приоритет
  - Высокий / Средний / Низкий (radio)
- История
  - Последние 10 запусков
  - Логи последнего запуска

### 4. Задачи (Task Manager)
**Список задач:**
- Заголовок
- Статус (todo/in_progress/done)
- Приоритет (high/medium/low)
- Исполнитель (агент)
- Дата создания
- Дата завершения
- Действия:
  - ✅ Завершить
  - ✏ Редактировать
  - 🗑 Удалить
  - 📋 Назначить агента

**Создание задачи:**
- Заголовок
- Описание
- Приоритет
- Назначить агента
- Срок выполнения
- Теги

### 5. Иерархия агентов (Org Chart)
**Визуализация:**
- Древовидная структура
- CEO → CTO → Devs
- CEO → PM → Tasks
- CEO → Security → Analysts
- Интерактивные узлы (клик = карточка агента)
- Анимации связей
- Статус каждого узла (цвет)

### 6. Бесплатные модели
**Меню выбора:**
- Список всех бесплатных моделей
- Фильтр по провайдеру
- Поиск по названию
- Информация о каждой модели:
  - Название
  - Размер (параметры)
  - Контекстное окно
  - Статус (активна/нет)
- Кнопка "Применить ко всем агентам"
- Кнопка "Лучшая модель" (автовыбор)

### 7. API ключи
**Управление ключами:**
- Список ключей (маскированные)
- Статус каждого ключа:
  - Активен / Заблокирован
  - Запросов сегодня
  - Токенов сегодня
  - Ошибок сегодня
- Кнопки:
  - ➕ Добавить ключ
  - 🔄 Ротировать
  - 🗑 Удалить
- Стратегия ротации:
  - Round Robin
  - Random
  - Least Loaded (по умолчанию)
- Автоматическая ротация (галочка)

### 8. Логи и мониторинг
**Журнал событий:**
- Время
- Агент
- Событие (запуск/стоп/ошибка)
- Детали
- Фильтры:
  - По агенту
  - По типу события
  - По времени
- Экспорт в CSV/JSON
- Автообновление (real-time)

### 9. Настройки системы
**Глобальные настройки:**
- Язык интерфейса (dropdown)
- Часовой пояс
- Тема оформления
- Провайдеры:
  - Добавить/удалить
  - API ключи
  - Base URL
- Уведомления:
  - Telegram бот
  - Email (будущее)
- Резервное копирование:
  - Автобэкап (галочка)
  - Интервал (часы)
  - Экспорт конфигурации

### 10. Мобильная версия
**Адаптивность:**
- Все экраны адаптивны
- Боковое меню → нижняя навигация
- Карточки → вертикальный список
- Графики → упрощенные
- Touch-friendly кнопки (мин 44px)

## ДАННЫЕ И API

### Источники данных
- REST API (FastAPI)
- WebSocket (real-time обновления)
- SQLite база данных

### Endpoints
- /api/health — статус системы
- /api/agents — список агентов
- /api/tasks — список задач
- /api/free-models — бесплатные модели
- /api/keys — API ключи
- /api/load-balancer — балансировщик
- /api/agent-config — настройки агентов
- /api/models/providers — провайдеры

- Lazy loading компонентов
- Кэширование API ответов
